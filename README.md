# Hospital Career Monitor

Hospital Career Monitor is a job intelligence platform designed to monitor hospital career pages and track recruitment activity across multiple healthcare organizations.

## Features

- Automated job scraping from hospital career portals
- Multi-hospital monitoring
- Job storage and tracking
- REST API using Flask
- React dashboard for analytics
- Hiring trend visualization
- Job count metrics by hospital
- Extensible parser architecture

## Tech Stack

### Backend
- Python
- Flask
- SQLite
- Requests
- BeautifulSoup

### Frontend
- React
- Recharts
- JavaScript

## Supported Hospitals

- Fortis Hospitals
- Apollo Hospitals
- Manipal Hospitals
- Max Healthcare
